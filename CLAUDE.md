# vikingmedia-site — Claude Code Context

## What this repo is
Static website for vikingmedia.org. Contains the landing page, Hugo briefings
site, and writing portfolio. Separated from the Skald pipeline repo so that
site content and pipeline code are independently versioned.

Deployed automatically to Cloudflare Pages on every push to main.

## Directory structure
```
/public/             Cloudflare Pages web root
/public/index.html   vikingmedia.org landing page
/public/images/      Shared images (logo, favicons)
/public/writing/     Nick Ferguson copywriting portfolio (static HTML + PDFs)
/site/               Hugo source for vikingmedia.org/skald/
/site/content/briefings/  Published briefing Markdown files
/site/themes/skald/  Hugo theme
/wrangler.toml       Cloudflare Pages config (assets directory: ./public)
```

## Key conventions
- `public/skald/` is gitignored — Hugo build output, generated by Cloudflare Pages on deploy
- Briefings are committed here by the HuggingFace Space dashboard (via GitHub API) when an
  editor approves and publishes. Do not edit briefing files manually unless backfilling.
- Static assets in `public/` (landing page, writing portfolio) are edited directly and pushed.
- Writing portfolio styles live in `public/writing/style.css` — do not inline them in index.html.

## Deployment
Cloudflare Pages build:
- Build command: `hugo --minify --source site`
- Deploy command: `npx wrangler deploy`
- Output directory handled by wrangler.toml (`directory = "./public"`)

Push to main → Cloudflare builds and deploys automatically. No manual step needed.

## Related repos

- HF Space `nick385/skald` — Gradio editorial dashboard; publishes briefings to this repo
